# Product Images

Add your product images here with the following naming convention:

- `product-1.jpg`
- `product-2.jpg`
- `product-3.jpg`
- etc.

Or use descriptive names like:
- `ethnic-suit-red.jpg`
- `lehenga-gold.jpg`
- `saree-maroon.jpg`

The images will be automatically loaded in the ProductCard component.
